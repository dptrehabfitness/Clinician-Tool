Database Setup Required

This app needs a free Firebase database so patients can log in from any device. Follow the steps below — it takes about 5 minutes.

Step 1: Go to console.firebase.google.com and click Add project
Step 2: Name it (e.g. dpt-rehab), disable Analytics, click Create project
Step 3: Click Firestore Database → Create database → Start in test mode → Done
Step 4: Click Project Settings (gear icon) → scroll to Your apps → click </> (Web)
Step 5: Register the app (any nickname), then copy the firebaseConfig values below

Your config is saved in your browser. You only need to do this once per device.
